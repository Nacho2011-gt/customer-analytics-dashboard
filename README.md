Customer Analytics Dashboard

Interactive customer analytics dashboard built in Power BI using DAX measures, customer segmentation, churn analysis, and business insights.

Project Overview

This project analyzes customer behavior in a global e-commerce business.

The dashboard includes:

Executive KPI overview
Customer segmentation analysis
Revenue analysis by category and loyalty tier
Churn analysis
Interactive filters and navigation
Business insights and recommendations
Tools Used
Power BI
DAX
Power Query
Data Visualization
Key Insights
Platinum customers generate the highest revenue and show the lowest churn rate.
Bronze customers present the greatest retention opportunity.
Beauty & Personal Care is the highest-performing product category.
Higher loyalty tiers correlate with higher customer spending.
